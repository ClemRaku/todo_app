# ✨ Flutter ToDo App ✨

## 📝 Introduction
Welcome to my simple ToDo application! 🚀 This is a straightforward and clean tool designed to help you keep track of your daily tasks without any fuss.

While this app is primarily targeted towards **Android** users 🤖, it works beautifully on **iOS** 🍏 as well, thanks to the magic of Flutter!

> [!NOTE]
> This app is designed with simplicity in mind to ensure a smooth user experience for everyone.

## 🌱 My Journey
I am currently on an exciting journey of **learning Flutter**! 📚 This is my very first project where I'm exploring the world of cross-platform mobile development.

Building this app has helped me understand:
- 🏗️ Flutter Widget Architecture
- 🎨 UI Design with Material Design
- 🔄 State Management basics

## ✨ Features
- **Simple Task Management**: Easily add and track your todos.
- **Cross-Platform**: Run on both Android and iOS devices.
- **Clean UI**: A bright and minimalist design.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/ClemRaku/todo_app.git
   ```
2. **Install dependencies**
   ```sh
   flutter pub get
   ```
3. **Run the app**
   ```sh
   flutter run
   ```
